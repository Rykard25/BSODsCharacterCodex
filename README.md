# BSOD's Character Codex v3

> A browser-based prompt builder for AI image generation with a massive character database, scene randomization, and model-specific presets.

## Features

### Character Database
- **244,932+ indexed characters** from anime, games, and manga
- Instant search by character name or series
- Auto-complete with popularity sorting
- Built-in tagging system (1girl, hair color, outfits, etc.)
- Import custom characters via TXT/CSV

### Prompt Engineering
- **Dual prompt boxes**: Positive and Negative tags
- **Syntax support**: A1111, NovelAI, Compel/ComfyUI, Plain text, Krea 2
- **Token estimation** with CLIP token counting
- Drag-and-drop tag reordering
- One-click copy with history (last 20 prompts saved)

### Model Presets
Quick-apply quality tags for specific models:
- **Pony** - Optimized for Pony Diffusion
- **Illustrious** - For Illustrious models
- **Anima** - Animagine XL specialized
- **SDXL** & **SD1.5** - Standard checkpoints

### Art Style Presets
20 curated styles including:
- Realistic, Semi-Realistic, Anime, Cartoon
- Painting, 3D Render, Pixel Art, Sketch
- Watercolor, Manga, Chibi, Concept Art
- Cyberpunk, Vaporwave, Claymation, Low Poly
- Vintage Photo, Pop Art, Ukiyo-e, Gothic

### Scene Builder
**600+ categorized tags** with randomization:
- **Expressions** (60+ tags) - smiles, angry, embarrassed, etc.
- **Clothing** (100+ tags) - dresses, uniforms, casual, armor
- **Actions** (80+ tags) - poses, activities, gestures
- **Locations** (100+ tags) - indoors, outdoors, fantasy settings
- **Lighting** (75+ tags) - golden hour, dramatic, soft, etc.
- **Composition** (65+ tags) - camera angles, framing
- **Objects** (100+ tags) - weapons, props, accessories

### Customization
**10 Built-in Themes**:
- Midnight (default cyan/dark)
- Synthwave (purple/pink)
- Terminal (green/black)
- Ember (orange/dark)
- Paper (light mode)
- Forest (green)
- Ocean (blue)
- Rose (pink)
- Solar (sepia light)
- Mono (grayscale)

### Additional Features
- **Notepad** with auto-save for jotting ideas
- **Audio controls** with Ambient/Piano/Lofi tracks
- **Quote widget** with dismissible inspiration quotes
- **Preset system** - Save/load prompt combinations
- **Import/Export** - Share presets as JSON
- **Responsive design** - Works on desktop and mobile

## Installation

No installation required! This is a single-file HTML application.
